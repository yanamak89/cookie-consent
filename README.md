# Cookie Consent

A simple cookie consent popup built with HTML, CSS, and JavaScript.

## Project URL
https://roadmap.sh/projects/cookie-consent

## Features
- Displays a cookie consent banner on page load
- Accept button hides the popup
- User consent is saved in localStorage
- Popup does not appear again after acceptance

## Technologies Used
- HTML
- CSS
- JavaScript
